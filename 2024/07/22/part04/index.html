vlookup

几个注意的点：

  1. 匹配的一般是第一个

  2. 0是精确匹配，1是模糊匹配

  3. 如果是查找a开头的

      &"*" /* 通配符 */

  4. 如果是查找a开头，往后数固定位数

      &"??" /* 2 */

  5. vlookup 查找的是符合的第一个来返回

  6. $ 来锁定函数里面的数值

match
