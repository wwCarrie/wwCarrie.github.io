# 打包成 EXE 说明

## 第一次打包（需要安装依赖）

1. **安装 PyInstaller**
```bash
pip install pyinstaller
```

2. **打包成 exe**
```bash
pyinstaller --onefile --console --name=git_push git_push.py
```

3. **exe 文件位置**
打包完成后，exe 文件在 `dist\git_push.exe`

## 后续重新打包（依赖已安装）

直接运行打包命令：
```bash
pyinstaller --onefile --console --name=git_push git_push.py
```

## 参数说明

- `--onefile`: 打包成单个 exe 文件
- `--console`: 显示控制台窗口（用于输入输出）
- `--name=git_push`: 指定输出文件名为 git_push.exe

## 测试

打包完成后，直接双击运行 `dist\git_push.exe` 即可。

---

**注意**：确保你的电脑上已安装 Git 并且已配置好 GitHub 认证。
