若干拉取方式和指令

  1. 写草稿

     hexo new draft + 草稿名

  2. 草稿要上线

     hexo publish + 草稿名

  3. 清理

     hexo clean

  4. 更新

     hexo g

  5. 运行

     hexo s

  6. 拉取到github上

     hexo d

  7. 更新且拉取github（4和6合并）

     hexo g --d
