序列

< 列表、元组、字符串 > – 序列

可以使用切片

列表

list = [1,2,3]

特性：

  1. 上限 2**63-1
  2. 混装
  3. 有序
  4. 允许重复
  5. 可以修改
元组

tuple = (1, 2, 3)

特性：

  1. 不可修改
  2. 如果是元组里嵌套了List，则可以修改
字符串

item = "hello"

特性：

  1. 无法修改
  2. 只能存储字符串
集合

set = {1,2,3}

特性：

  1. 无序 （不支持下标访问）
  2. 允许修改
  3. 去重
字典

dict = {"苏嘉莉":99, "苏小弟":98}

特性：

  1. 不能用下标，但能用Key，不能while
  2. 重复后，新盖旧
  3. 可以是任意数据类型，但Key不能是字典
